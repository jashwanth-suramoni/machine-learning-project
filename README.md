# Transaction Fraud Detection using Machine Learning

## 📌 Project Overview
This project develops a Machine Learning model to detect fraudulent bank transactions. The model analyzes transaction details and classifies them as **Fraudulent** or **Legitimate**, helping financial institutions reduce financial losses and improve customer security.

---

## 🎯 Business Problem
Financial institutions process millions of transactions daily. Detecting fraudulent transactions manually is time-consuming and inefficient. This project provides an automated fraud detection system using Machine Learning to improve accuracy and enable real-time fraud detection.

---

## 🎯 Business Objective
- Build a Machine Learning model to classify transactions as Fraudulent or Legitimate.
- Improve fraud detection accuracy.
- Reduce financial losses caused by fraud.
- Support fast and reliable transaction monitoring.

---

## ⚠ Business Constraints
- Fraudulent transactions are much fewer than legitimate ones (class imbalance).
- Missing fraudulent transactions (False Negatives) leads to financial loss.
- Incorrectly flagging genuine transactions (False Positives) affects customer experience.
- The model should provide quick predictions suitable for real-time applications.
